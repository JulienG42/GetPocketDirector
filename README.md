# Get Pocket Director

Public distribution repository for Pocket Director updates.

This repository is intentionally separate from the private Pocket Director source repository. It contains only files that need to be reachable by users and by Sparkle:

- the public Sparkle `appcast.xml` feed;
- selected release archives published through GitHub Releases;
- release notes or download information when needed.

A development build is not a public update. Pocket Director only offers versions that are deliberately published in `appcast.xml`.
